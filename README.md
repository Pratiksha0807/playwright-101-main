# Web Automation With Playwright On LambdaTest Cloud

## Pre-requisites

 * Node.js  (Node.js 18.x and above is recommended) 
 * Feel free to use any OS
 * Windows - Windows 10+, Windows Server 2016+ or Windows Subsystem for Linux (WSL).
 * MAC - MacOS 12 Monterey, MacOS 13 Ventura, or MacOS 14 Sonoma.
 * Linux - Debian 11, Debian 12, Ubuntu 20.04 or Ubuntu 22.04, with x86-64 or arm64  architecture.
 * Git - Any version
 * Gitpod

## Getting Started

[Create Your Own LambdatTest Account](https://accounts.lambdatest.com/register)

[Clone This Repository](https://github.com/kkr313/playwright-101.git)

[Capability Generator](https://www.lambdatest.com/capabilities-generator/)

```
git clone https://github.com/kkr313/playwright-101.git
```

## Execute the test

Get your username and access key from lambdatest and replace in "capabilities.js" file

You can get username and access key from capability generator or from account settings.

Set LT_USERNAME and LT_ACCESS_KEY in environment variable </br></br>
*Windows*</br>
set LT_USERNAME="YOUR_LAMBDATEST_USERNAME"</br>
set LT_ACCESS_KEY="YOUR_LAMBDATEST_ACCESS_KEY" 

</br>

*MAC and Linux* </br>
export LT_USERNAME="YOUR_LAMBDATEST_USERNAME" </br>
export LT_ACCESS_KEY="YOUR_LAMBDATEST_ACCESS_KEY" </br>

## Execute below command

```
    cd playwright-101

    To run the test
        npx playwright test

    To open the Report -
        npx playwright show-report"
```
